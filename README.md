# RI Cluster Analyzer
Торговая система кластерного анализа для фьючерса на индекс РТС (RI) на Московской бирже. Состоит из четырёх независимых агентов: сбор данных, анализ сигналов, визуализация и новостной анализ через LLM.
## Возможности
- Кластерный анализ обезличенных сделок (футпринт-график) с разбивкой по Bid/Ask
- Детектор паттернов: Absorption, Imbalance, Exhaustion
- Расчёт ключевых уровней предыдущего дня: POC, VAH, VAL
- Индикатор VWAP с сигма-каналами (±1σ, ±2σ, ±3σ)
- Корреляционный анализ с Brent и USD/RUB
- Скоринговая система генерации сигналов Long/Short с ATR-стопами и R:R = 1:2
- Интерактивная веб-панель на Plotly Dash с реалтайм-обновлением
- Новостной анализ через YandexGPT с парсингом RSS и Telegram-каналов
- Автоматический запрет направления при сильном негативном/позитивном новостном фоне
## Архитектура
Система состоит из четырёх агентов, работающих параллельно:
| Агент 	    | Файл 			            | Назначение |
|-------	    |------			            |------------|
| Reader   	  | `reader.py` 		      | Подключение к QUIK, сбор обезличенных сделок, формирование кластеров |
| Analyzer 	  | `analyzer.py` 	      | Анализ кластеров, детектор паттернов, генерация сигналов |
| Visualizer 	| `visualizer.py`   	  | Веб-панель на Plotly Dash (http://localhost:8050) |
| News Analyzer | `news_analyzer.py` 	| Парсинг новостей RSS+Telegram, LLM-анализ через YandexGPT |

Подробное описание архитектуры — в [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md).

Запуск

Агент 1: Reader — сбор данных из QUIK
Агент 2: Analyzer — генерация сигналов
Агент 3: Visualizer — открывает браузер на
<a href="http://localhost:8050">http://localhost:8050</a>
Агент 4: News Analyzer — анализ новостей
